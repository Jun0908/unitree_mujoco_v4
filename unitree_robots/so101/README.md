# SO-101 Model

This directory contains the LeRobot SO-101 follower arm model for MuJoCo display.

- `so101_follower.urdf`: URDF model with mesh paths adjusted for this repository
- `meshes/`: STL assets used by the URDF
- `LICENSE.so101-ros-physical-ai`: upstream Apache-2.0 license

Source:
